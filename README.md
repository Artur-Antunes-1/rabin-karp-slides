# Rabin–Karp · Slides (Teoria da Computação)

Apresentação sobre o algoritmo **Rabin–Karp** — busca de padrão em texto e análise de
complexidade de tempo, com medições reais comparando **C** e **Python**.

### ▶️ Ver os slides (ao vivo)

**https://artur-antunes-1.github.io/rabin-karp-slides/**

### Como navegar
- Setas `←` `→` ou barra de espaço para avançar/voltar
- `F` para tela cheia · `Home` / `End` para ir ao início/fim
- Para gerar PDF: `Ctrl+P` → "Salvar como PDF" → layout **paisagem**

### Sobre
- Deck em HTML único e autossuficiente (funciona offline; as fontes vêm do Google Fonts quando há internet).
- Gráficos desenhados em SVG a partir dos dados reais (médias de 30 rodadas, n = 2.000 / 20.000 / 100.000, m = 16).
- Código, dados e relatório do projeto: https://github.com/ricardomvlins/rabin_karp
